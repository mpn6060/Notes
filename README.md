# NotesSolution - ASP.NET Core 8 Notes App (LocalDB + Bootstrap 5)

This repository contains a minimal, runnable scaffold for a Notes application using Clean Architecture:
- Notes.Domain
- Notes.Application
- Notes.Infrastructure (EF Core + SQL Server LocalDB)
- Notes.Web (ASP.NET Core MVC + Bootstrap 5 + Identity wiring)

See `src/Notes.Web/README_RUN.md` for run instructions.
